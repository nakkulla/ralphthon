# Image Prompting

Prompt goal: one representative project image, not a UI screenshot.

Include:
- project domain and core value;
- 2-4 visual metaphors;
- clean app-icon or poster composition;
- avoid logos/trademarks unless user supplied them.

Template:

```text
A modern app-icon style emblem for <project>, representing <domain/value> with <visual metaphors>, clean composition, high contrast, no text, no logos.
```

After `image_gen`, inspect the generated image/description and add 2-5 concrete keywords such as `waveform`, `gradient`, `dashboard`, or `map` before `pgal profile create`.
